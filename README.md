# ПО «Каталог продуктов»

## Особенности реализации

1. Содержит таблицы для каталога продуктов.
2. Механизм аутентификации и авторизации с 3 ролями:
    -  Простой пользователь;
    -  Продвинутый пользователь;
    -  Администратор.
3. Механизм логгирования.
4. Механизм удаления/добавления/изменения категории в соответствующем справочнике.
5. Механизм добавления/изменения/удаления продукта. Каскадное удаление продукта при удалении категории.
6. Возможность поиска/фильтрации общего каталога продуктов. Выделение товаров определенной категории.
7. Конвертация валюты в доллар США (USD) при наведении на символ `*` без перезагрузки основной страницы. 
8. Клиентский интерфейс: Web-приложение (**ASP.Net Core MVC**, **HTML**, **CSS**, **JS**).
9. Взаимодействие с серверной частью через **Web API**. СУБД - **MS SQL Server**.  
10. При работе с базой данных реализован паттерн **Repository** (DAO).
